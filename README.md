# Customer Churn Analysis

## Business Problem
Customer churn negatively impacts revenue. This project analyzes customer behavior to identify churn drivers and propose data-driven retention strategies.

## Tools Used
- Excel
- SQL
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Tableau

## Key KPIs
- Churn Rate
- Retention Rate
- Average Tenure
- Monthly Revenue Loss

## Key Insights
- Month-to-month contracts have the highest churn
- New customers (<12 months) are most likely to churn
- Higher monthly charges correlate with churn
- Auto-pay users churn less

## Business Recommendations
- Incentivize long-term contracts
- Improve onboarding for first 6 months
- Promote auto-pay discounts
- Target high-risk customers with retention campaigns
